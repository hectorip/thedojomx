¿Quieres aprender a programar?

Aprende a programar creando una página web completamente funcional, atacando una necesidad real. Aprende las bases de HTML, CSS y JavaScript.


Aplica al programa

Acerca del programa


Aprenderás las bases de la programación y prácticas modernas de desarrollo de software mediante la creación de un proyecto real,
apoyando a una ONG o a un proyecto con un fin social.

Entiende como se crea un proyecto real

Entenderás todo el flujo de trabajo de un proyecto real, desde la idea hasta ponerlo en marhca para los usuarios finales.
Aprenderás todo el lenguaje de los programadores y la manera de comunicarte con ellos (o convertirte en uno).

Adquiere las habilidades para hacerlo por tu cuenta

Aprenderás la lógica básica para programar y continuar creciendo, además de tres lenguajes indispensables para el desarrollo web actual: HTML, CSS, JavaScript .

Ayuda a una causa social

Este no es un curso teórico. Lo que aprendas será aplicado inmediatamente a un proyecto, provisto por una institución con una necesidad real que atenderemos mediante un proyecto web.
